Поиск по полученным результатам

Можно заполнить сразу несколько полей или только одно.

Простой поиск

Задайте полную форму имени, для которого вы хотите посмотреть список предложенных респондентами вариантов.

Дополнительные опции

Если вы хотите посмотреть ответы для имени из конкретного региона, укажите его в поле ниже. Список регионов, для которых доступны хотя бы какие-то ответы, можно найти на странице "Статистика".

Для того, чтобы посмотреть, как сокращают имена люди определённого возраста, укажите его в поле ниже:

до и включая указанный возраст
после и включая указанный возраст
ровно указанного возраста, ни больше, ни меньше

Поиск по диминутиву

Если вас интересует конкретный диминутив, введите его в поле ниже. Возможно, его не существует… Если всё-таки существует, то выдача расскажет о людях, употребивших его — возраст, регион проживания и так далее.
